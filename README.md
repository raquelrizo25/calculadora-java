# 🧮 Calculadora en Java

Este proyecto es una aplicación de consola desarrollada en Java que permite realizar operaciones básicas:

- Suma
- Resta
- Multiplicación
- División

## 🚀 Cómo funciona

El programa solicita al usuario:
1. Dos números
2. Una operación (+, -, *, /)

Y muestra el resultado por pantalla.

## 🛠️ Tecnologías utilizadas

- Java
- Scanner (entrada de datos)

## 📚 Lo que he aprendido

- Uso de variables y tipos de datos
- Control de flujo con switch
- Manejo de errores (división entre 0)
